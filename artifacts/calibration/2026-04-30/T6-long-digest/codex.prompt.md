# Release Notes Synthesis

Read CHANGELOG.md (full file, covers v0.27.0 through v0.42.0).

Write a 500-word "release notes for stakeholders" document suitable for
non-engineers: managers, academic collaborators, lab directors.

Identify major themes and the project's narrative arc.

## Output format
500 words, narrative paragraphs (NOT bullet lists). Accessible language.
Group by themes, not by version number. Hard cap: exactly 500 words.
