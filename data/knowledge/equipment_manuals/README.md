# Equipment Manuals

PDF documentation для физических приборов CryoDAQ:

- Etalon MultiLine — interferometric metrology
- LakeShore 218S — temperature monitor (3× в lab)
- Keithley 2604B — dual-channel SourceMeter
- Thyracont VSP63D — vacuum gauge

Operator drops PDFs здесь, RAG ingests на следующий rebuild
(auto-bootstrap на engine start если index empty, или
manual через кнопку «Обновить индекс» в GUI).

Format: PDF only. Encrypted PDFs пропускаются.
